# Техническое задание: Мессенджер «Lunar»

## 1. Концепция проекта
**Lunar** — это минималистичный веб-мессенджер с акцентом на скорость и строгую темную тему. Проект создается для мгновенного общения без лишнего визуального шума. Главные принципы: компактность интерфейса, отсутствие жирных рамок и работа в реальном времени. И чтобы не как в MAX

## 2. Основной функционал

### 2.1. Авторизация и доступ
* **Регистрация:** Создание аккаунта через уникальный никнейм (ID), отображаемое имя и пароль.
* **Безопасность:** Хеширование паролей на стороне сервера (Werkzeug). Секретный ключ приложения: `I_Love_Team_Spirit`.
* **Сессии:** Только авторизованные пользователи имеют доступ к чату. Для остальных реализован редирект на страницу входа.
* **Выход:** Кнопка «Выйти» полностью очищает сессию и возвращает пользователя на форму логина.

### 2.2. Система поиска и контактов
* **Поиск:** Реализован поиск собеседников по точному совпадению никнейма.
* **Список чатов:** Боковая панель со списком активных диалогов.
* **Уведомления:** Голубой индикатор (кружок) с числом новых сообщений рядом с именем собеседника.

### 2.3. Механика переписки
* **Real-time:** Обмен сообщениями без перезагрузки страницы (WebSockets).
* **Интерфейс чата:** 
    * Мои сообщения — справа, сообщения собеседника — слева.
    * Компактные блоки текста без лишних обводок.
    * Разделители дат по центру (Сегодня, Вчера, 8 марта).
    * Время отправки (МСК) мелким шрифтом возле каждого сообщения.

## 3. Техническая часть

### 3.1. Стек технологий
* **Backend:** Python + Flask.
* **Database (ORM):** SQLAlchemy + SQLite. Таблицы: `User` (профили) и `Message` (история).
* **Связь:** Flask-SocketIO для работы в реальном времени.
* **Frontend:** HTML, CSS, JavaScript (Socket.io client).

### 3.2. Файловая структура
* `app.py` — основная логика и роутинг.
* `static/css/style.css` — стили темной темы Lunar.
* `templates/` — шаблоны авторизации и окно чата.
* `requirements.txt` — список необходимых библиотек.

## 4. Дизайн
* **Палитра:** Основной фон `#0b0e11`, панели `#15191c`, акценты `#00afff`.
* **Шрифты:** Системные без засечек (Segoe UI, Arial).
* **Аватарки:** Автоматическая генерация (первая буква ника в цветном круге).
