# Техническое задание: Мессенджер «Lunar»

### Основные функциональные требования

**1) Авторизация и профили**
* Регистрация через уникальный никнейм, отображаемое имя и пароль.
* Вход по нику и паролю. Пароли хранятся в хешированном виде (pbkdf2:sha256).
* Функция выхода из системы с возвратом на страницу авторизации.
* Автоматическое создание аватарки (первая буква имени) с возможностью загрузки своего изображения.

**2) Список диалогов и поиск**
* Боковая панель со списком активных чатов, отсортированных по дате свежего сообщения.
* Живой поиск пользователей по точному совпадению никнейма.
* Индикация непрочитанных сообщений (синий кружок с числом).
* Автоматический сброс счетчика непрочитанных при открытии чата.

**3) Переписка в реальном времени**
* Реализация мгновенной доставки сообщений через веб-сокеты без обновления страницы.
* Визуальное разграничение: сообщения пользователя — справа (голубые), собеседника — слева (серые).
* Отображение времени отправки (МСК) и разделителей дат (Сегодня, Вчера).
* Возможность вернуть последнее сообщение в поле ввода по нажатию стрелки "Вверх".
* Статус прочтения сообщений: одна галочка (✓) — доставлено, две галочки (✓✓) — прочитано.
* Возможность удаления своих сообщений через контекстное меню (правый клик мыши), после удаления у всех участников чата сообщение заменяется на текст "Удалено".
* Индикатор "печатает..." отображается, когда собеседник набирает текст.
* Отображение статуса "онлайн" (зелёная точка рядом с именем пользователя).

**4) Файлы и вложения**
* Возможность отправки файлов любого типа через кнопку прикрепления (+).
* Отображение файлов в чате в виде кликабельных ссылок для скачивания.
* Механизм загрузки и смены аватара профиля через серверную папку static/uploads.

# Техническое задание: Мессенджер «Lunar»

### Основные функциональные требования

**1) Авторизация и профили**
* Регистрация и вход по уникальному никнейму и паролю (хеширование pbkdf2:sha256).
* Автоматическая генерация аватара по первой букве имени с возможностью загрузки своего изображения.
* Функция выхода из системы с перенаправлением на страницу авторизации.

**2) Список диалогов и поиск**
* Боковая панель со списком активных чатов, отсортированных по времени последнего сообщения.
* Живой поиск пользователей по точному совпадению никнейма.
* Индикация непрочитанных сообщений (синий круг со счетчиком) и их автоматический сброс.

**3) Переписка в реальном времени (WebSockets)**
* Мгновенная доставка сообщений без обновления страницы.
* Визуальное разграничение: свои сообщения — справа (голубые), собеседника — слева (серые).
* Статусы сообщений: доставлено (✓) и прочитано (✓✓).
* Индикатор «печатает...» и отображение статуса «онлайн» (зелёная точка).
* Удаление своих сообщений через контекстное меню (правый клик) с заменой текста на «Удалено».

**4) Файлы и вложения**
* Отправка файлов любого типа через кнопку (+).
* Отображение вложений в чате в виде кликабельных ссылок для скачивания.

---

### Архитектура и дизайн

**Основные модули**
* **Главное окно (chat.html):** Основной интерфейс, список контактов и активная область переписки.
* **Окно авторизации (auth.html):** Формы входа и регистрации.
* **Серверная логика (app.py):** Flask-SocketIO, управление сессиями и маршрутизация.
* **Модуль базы данных (models.py):** SQLAlchemy (SQLite). Таблицы:
    * *User:* id, username, display_name, password, avatar, is_online.
    * *Message:* id, sender_id, recipient_id, content, file_name, file_data, is_read, timestamp (МСК).

**Дизайн и интерфейс**
* **Тема:** Тёмная (фон `#0b0e11`, панели `#15191c`, акцент `#00afff`).
* **Стиль:** Минимализм, скругления 8-12px, тонкие скроллбары (4px).
* **Шрифт:** Системный без засечек (Segoe UI, Arial).

**Горячие клавиши**
* **Enter:** Отправить сообщение.
* **Стрелка вверх:** Вернуть последнее отправленное сообщение в поле ввода.
* **Esc:** Убрать фокус с поля ввода.

**Технические детали**
* **Секретный ключ:** `Lunar_Yandex_Lyceum_2026`.
* **Хранение:** БД SQLite в папке `instance/`, аватары в `static/uploads`, файлы в БД (BLOB).

# Пояснительная записка к проекту «Lunar»

### 1. Описание файлов

*   **`app.py`** — Основной файл сервера. Содержит настройки Flask, обработку WebSocket-событий (отправка сообщений, уведомления о печати, статусы онлайн) и логику авторизации.
*   **`models.py`** — Описание таблиц базы данных SQLAlchemy: `User` (данные профиля) и `Message` (текст, файлы, время и статус прочтения).
*   **`requirements.txt`** — Список Python-библиотек, необходимых для работы проекта.
*   **`static/`** — Папка для хранения статических данных:
    *   `css/style.css` — Оформление интерфейса: тёмная тема, стили сообщений и адаптивная верстка.
    *   `uploads/` — Папка для загруженных аватарок и переданных файлов.
*   **`templates/`** — Шаблоны страниц:
    *   `auth.html` — Формы регистрации и входа.
    *   `chat.html` — Главный экран мессенджера (интерфейс чата и клиентская JavaScript-логика работы с сокетами).
*   **`instance/`** — Директория с файлом базы данных `lunar.db` (создается автоматически).

---

### 2. Инструкция по запуску

1.  **Установка модулей:**
    Выполните команду в терминале:
    ```bash
    pip install -r requirements.txt
    ```

2.  **Запуск проекта:**
    Запустите сервер:
    ```bash
    python app.py
    ```

3.  **Работа в браузере:**
    Перейдите по адресу: `http://127.0.0.1:5000`

---

### 3. Технические особенности
*   **Логика JS:** Весь интерактив (обработка сокетов, горячие клавиши, прокрутка чата) находится внутри файла `chat.html`.
*   **Безопасность:** Пароли хранятся в зашифрованном виде.
*   **Файлы:** Реализована автоматическая очистка или уникальное именование файлов в `uploads` для исключения конфликтов.
